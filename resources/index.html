Resources

A curated list of resources I've found genuinely useful — especially for first-gen students, underrepresented scholars, and anyone figuring out how academia works.

Small note: These resources are the things I have either used myself or wish I wish I'd known about earlier. If you would like to contribute something that helped you, send it my way — I'd love to keep this list growing.

Programming

  • 🌀 Swirl Free resource that teaches you R right in R Studio.
  • 🏫 Code Academy Code Academy has helpful courses for foundational knowledge in programming and data science. They also offer financial aid and discounts for students or individuals of low-income backgrounds.
  • 🏎️ fasteR: Fast Lane to Learning R! Online course by Professor Norm Matloff.

Design

  • 🎨 Figma Free for students. All my vector art for web applications are done in Figma.
  • 🖼️ Canva Canva Pro is also free for students. It's the most user friendly option for those without much graphic design experience.
  • 🖌️ Procreate Great option if you have an iPad. Their UI is straightforward and it's easy to make simple animated gifs using Procreate.
  • 📐 Affinity Designer More affordable alternative to Adobe Illustrator (bought by Canva).

Research

  • 📚 Zotero Free, open-source reference manager. A non-negotiable for literature reviews.
  • 🕸️ Connected Papers Visualizes the citation network around a paper — great for finding adjacent literature fast.
  • 👩🏽‍🔬 FLi Sci Non-profit program for under-served high school students to get involved in research early led by Gabriel Reyes (who helped me to get my start in research!)
  • 🌉 Columbia Bridge to the Ph.D. For those who can't afford to do a Master's in order to pursue research, this is an excellent paid post-bac opportunity.
  • 🦾 Pittsburgh Hot Metal Bridge Post-Bac Program Another great post-bac program that provides a stipend and housing for fresh graduates
